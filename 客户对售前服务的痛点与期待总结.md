**核心洞察**：客户在软件外包过程中遇到的80%的问题，都源于**售前阶段的需求不明确**。而这些问题，都可以通过高质量的售前交付物（需求文档、原型图、技术方案）来直接缓解。

## 专业软件公司/工作室的典型售前流程与交付物

通过对猪八戒网、知乎等平台上专业软件公司和工作室的售前流程进行分析，我们发现，专业的售前服务通常包含以下流程和交付物：

### 典型售前流程

1. **初步沟通**：了解客户的业务背景、目标和预算。
2. **需求调研**：通过访谈、问卷等方式，深入了解客户的具体需求。
3. **方案设计**：根据需求，设计技术方案、产品原型和报价方案。
4. **方案讲解**：向客户讲解方案，解答疑问，并根据反馈进行修改。
5. **签订合同**：确认方案后，签订售前服务合同或开发合同。

### 典型售前交付物

| 交付物 | 价值 | AI赋能点 |
| :--- | :--- | :--- |
| **需求文档** | 明确项目范围，避免后期扯皮 | ✅ GPT生成初稿 |
| **原型图** | 可视化需求，降低沟通成本 | ✅ Figma AI、Midjourney |
| **业务流程图** | 梳理业务逻辑，发现潜在问题 | ✅ Mermaid、Draw.io |
| **技术方案** | 评估技术可行性，给出合理报价 | ✅ GPT生成技术选型建议 |
| **报价说明** | 详细说明报价构成，增加透明度 | ✅ GPT生成报价模板 |

**核心洞察**：专业公司非常重视售前阶段的交付物，因为这些交付物是建立信任、降低风险、提高转化率的关键。而这些交付物，恰恰是你的"AI技术顾问"模式最擅长生成的。



## 市场上“需求分析/原型/方案”类服务的现状与定价

通过对猪八戒网、闲鱼等平台的调研，我们发现，市场上“需求分析/原型/方案”类服务的现状如下：

### 猪八戒网：专业但价格不透明

- **服务内容**：提供需求分析、原型设计、技术方案等全套售前服务。
- **定价模式**：多为“按项目收费”，但价格不公开，需要私聊。
- **典型案例**：8年经验的产品经理，按小时收费为¥100/小时。

### 闲鱼：市场空白，机会巨大

- **服务内容**：几乎没有单独售卖“需求分析”或“原型设计”服务的案例。
- **主要商品**：低价的模板文档（¥1-3元）和高价的全流程开发服务（¥1000+）。
- **市场机会**：闲鱼上缺少“售前服务”的产品化供给，这正是新手中介的机会所在。

### UI设计行业：可作为定价参考

- **普通商业需求**：¥300 - ¥800/页（能出高保真原型图）。
- **资深外包项目**：¥1000+/页（带高保真原型图）。

**核心洞察**：售前服务的定价，既要体现专业价值，又要降低客户的决策门槛。参考UI设计行业的定价，一个包含10-20页原型图和需求文档的售前服务包，定价在¥1000-2000元是比较合理的。



## 适合「AI 技术顾问中介」的售前产品化服务设计建议

基于以上分析，我们为您设计了2个适合在闲鱼上架的“AI技术顾问式售前套餐”，并给出了详细的起步策略建议。

### 服务套餐一：「小程序/软件需求深度梳理包」

- **服务名称**：「小程序/软件需求深度梳理包」
- **适用客户**：有想法但不懂技术的老板、被报价吓到的客户、之前被坑过的客户。
- **核心交付物**：
  - 《需求访谈记录》（AI整理录音）
  - 《功能需求清单》（AI生成初稿）
  - 《业务流程图》（AI辅助绘制）
  - 《技术实现建议》（AI生成初稿）
  - 《外包开发避坑指南》（AI生成模板）
- **推荐定价**：¥600 - ¥1500（根据交付物深度分级），后续开发可抵扣50%。
- **价值卖点**：省钱（避免返工）、省时（提高效率）、省心（多方比价）、避坑（控制成本）、专业（经验分享）。

### 服务套餐二：「小程序/软件原型设计包」

- **服务名称**：「小程序/软件原型设计包」
- **适用客户**：需求明确但不懂设计的客户、想先看效果再开发的客户、需要原型融资的客户。
- **核心交付物**：
  - 《产品原型图》（Figma AI、Midjourney辅助）
  - 《页面功能说明》（AI生成）
  - 《技术实现建议》（AI生成）
- **推荐定价**：¥800 - ¥3000（根据页面数量分级），后续开发可抵扣50%。
- **价值卖点**：看得见（可视化）、可修改（成本低）、好沟通（无歧义）、降风险（提前发现问题）、融资利器（更具说服力）。

### 推荐的起步策略

1. **第一阶段（0-3个月）**：主推套餐一「需求深度梳理包」，风险最低，交付最简单，快速积累案例和口碑。
2. **第二阶段（3-6个月）**：增加套餐二「原型设计包」，客单价更高，利润空间更大，建立“专业形象”。
3. **第三阶段（6个月以后）**：尝试「需求梳理 + 原型设计 + 外包开发协助」全包套餐，客单价最高，客户粘性最强，实现业务升级。

**核心洞察**：通过产品化的售前服务，您可以将自己的专业能力和AI工具的效率优势，转化为客户看得见的价值，从而在闲鱼上建立起“降维打击式”的竞争优势。
